Описание системы.
Проект представляет собой систему управления световыми параметрами с использованием трех флекс-сенсоров. Устройство построено на базе микроконтроллера ESP32 и включает в себя передатчик, установленный на перчатке, и приемник с RGB-лентой. 
Передающее устройство питается от литий-ионного аккумулятора и устанавливается на перчатке. Приемное устройство с RGB-лентой из 80 светодиодов получает питание от пауэрбанка. Связь между устройствами осуществляется через Wi-Fi сеть с использованием MQTT протокола для передачи управляющих команд.

Назначение сенсоров.
Каждый из трех флекс-сенсоров выполняет строго определенную функцию. Первый сенсор отвечает за выбор цвета светового потока. Второй сенсор контролирует уровень яркости свечения. Третий сенсор обеспечивает фиксацию текущего режима работы системы.

Процедура калибровки.
При каждом включении системы выполняется автоматическая процедура калибровки всех трех флекс-сенсоров. Калибровка определяет базовые значения для разогнутого и согнутого состояния каждого пальца. Это обеспечивает точное соответствие между углом изгиба пальца и управляемым параметром.

Режим работы.
После успешной калибровки система переходит в рабочий режим. Пользователь может изменять цвет свечения, регулировать яркость и фиксировать понравившиеся настройки с помощью соответствующих движений пальцев. Все изменения применяются в реальном времени к RGB-ленте приемного устройства.
